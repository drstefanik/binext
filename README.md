# BI NEXT Frontend v3 (Tailwind)

- Tailwind + PostCSS inclusi
- Home su `/`, Login su `/login`
- SPA rewrites: `vercel.json`
- Env: `VITE_AUTH_API`

## Setup
```bash
npm install
cp .env.example .env
npm run dev
npm run build
npm run preview
```

## Vercel
- Build: `npm run build`
- Output: `dist`
- Env: `VITE_AUTH_API` (Production/Preview)
